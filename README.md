# SDET_Fancode

## Prerequisites

- Node.js
- npm

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/rufus-619/SDET_Fancode.git
    ```

2. Install the dependencies:
    ```sh
    npm install
    ```

## Usage

To run the main application:
```sh
node index.js

## Testing

To run the tests:
```sh
npm test
